Acerca de

Dashboard Unison COVID-19 Sonora

Este tablero es un recurso elaborado por un grupo de profesores del Departamento de Matemticas de la Universidad de Sonora del rea de Ciencia de Datos. El tablero se realiz en el lenguje R usando el lenguaje de marcado Rmarkdown y la plantilla flexdashboard for R. Nos basamos en un ejemplo base desarrollado por Rami Krispin y el cual se puede consultar aqu.

Datos

Los datos sobre el proceso del COVID-19 en el Estado de Sonora se reciben diariamente de la Secretara de Salud del Estado de Sonora, gracias a la colaboracin entre la Universidad de Sonora y el Gobierno del Estado durante este periodo excepcional. Los datos sobre el proceso del COVID-19 de los estados de Mxico se tomaron del repositorio de datos manenido por Gabriel Alfonso Carranco-Sapins. Esta base se actualiza cada da a partir de la informacin de la SSA. Los datos sobre la poblacin de Mxico se tomaron de INEGI. Los datos sobre el proceso del COVID-19 en el estado Arizona (EU) se tomaron del Centro de recursos sobre Coronavirus de la Universidad Johns Hopkins University & Medicine. Esta base de datos se actualizan cada da a media noche y se pueden obtener aqu.

Paquetes utilizados y documentacin

  • Tablero: flexdashboard
  • Grficas: plotly
  • Mapa: leaflet
  • Manipulacin de datos: dplyr tidyr
  • Tablas: DT

Reproducibilidad

El tablero se genera diariamente como una pgina web esttica. Es posible realizar un fork al proyecto en Github, descargarlo, modificarlo y publicar un tablero similar en las pginas web que desees.
